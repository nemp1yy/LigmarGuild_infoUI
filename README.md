# 🏰 LIGMAR Guild Database Manager

Приложение для управления гильдией в телеграм-игре LIGMAR. Позволяет отслеживать игроков, их активность, вклад в гильдию и участие в событиях.

## ✨ Основные возможности

- **Управление игроками**: добавление/удаление, редактирование профилей, просмотр информации
- **Аналитика активности**: учет урона, участие в рейдах, мониторинг вклада
- **Система событий**: учет участия, история активности
- **Расширенный поиск**: фильтрация по уровню, классу, статусу, датам

## ⚙️ Технологии

- Python 3.9+
- PyQt6
- SQLite
- MVC архитектура

## 🚀 Запуск

1. Установите зависимости:
```bash
pip install PyQt6
```

2. Запустите приложение:
```bash
python MainWindow.py
```
